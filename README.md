# 🍔 Swiggy AI Food Delivery Analytics System

This project is an AI-powered data science system inspired by the Swiggy food delivery platform. 
It analyzes food delivery data and uses machine learning and deep learning models to generate 
insights, predictions, and recommendations.

The system integrates multiple AI modules including:

• Restaurant Recommendation System  
• Delivery Time Prediction Model  
• Customer Segmentation using K-Means  
• Demand Forecasting using LSTM  
• Interactive Analytics Dashboard built with Streamlit  

The project demonstrates how AI and data science can improve operational efficiency 
in food delivery platforms by predicting delivery times, understanding customer behavior, 
and recommending restaurants.

This project is designed as a Data Science / AI portfolio project suitable for:

• Machine Learning internships  
• Data Science portfolios  
• Final year engineering projects  
• AI product prototypes
